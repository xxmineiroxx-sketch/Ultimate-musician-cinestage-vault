# Product Roles & Apps

## User types
- Admin / Music Director (Organizer)
- Worship Team Member (Playback)
- Crew Member (Playback): FOH, Lights, ProPresenter, Stream

## Core concepts
- Organization/Team
- Service (date, type)
- Setlist (ordered)
- Role (Acoustic, Electric, Keys, BGV1, Drums, FOH…)
- Assignment (role ↔ member per service)
- Arrangement (master)
- Instrument chart (derived per role)
