# LIVE MODE v2.8 – AutoMatch + Run Service

## AutoMatch
Automatically matches:
- Song → BPM
- Song → Key
- Song → Default cues
- Song → Mixer snapshot

## Run Service Mode
- Locks service order
- Disables editing
- Enables large NEXT button
- Activates Sticky Sync

## Fail Conditions
- Missing stems → fallback stereo
- Missing click → silent click
- Missing guide → auto disable

## Goal
Zero thinking during service.
Only execution.
