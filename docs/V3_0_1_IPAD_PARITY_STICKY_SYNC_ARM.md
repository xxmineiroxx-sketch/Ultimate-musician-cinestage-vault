# V3.0.1 – iPad Parity + Sticky Sync ARM

## Sticky Sync
A soft-lock sync system between devices.

## ARM State
- Organizer ARMS changes
- Stage RECEIVES
- Live EXECUTES

## Rules
- Last ARM wins
- Offline device replays last ARM
- No destructive overwrites

## Devices
- iPad (Organizer)
- iPad (Stage)
- iPhone (Live optional)
