# LIVE MODE v2.7 – Service Plan Cue Grid

## Purpose
High-level grid view of an entire service with cues per item.

## Grid Layout (iPad Landscape)
| Order | Item | Key | BPM | Cues | Status |
|------|------|-----|-----|------|--------|

## Color Coding
- Gray: Planned
- Blue: Armed
- Green: Completed
- Red: Error / Skipped

## Behaviors
- Tap row → open Cue Editor
- Long press → reorder service
- Swipe right → mark completed

## Permissions
- Organizer: Full edit
- Stage: View only
- FOH: View only

## Sync
- Grid is source of truth
- Pushes to Stage & Live instantly
