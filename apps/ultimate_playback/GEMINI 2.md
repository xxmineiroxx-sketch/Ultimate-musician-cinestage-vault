# 🎹 ULTIMATE PLAYBACK (MUSICIAN APP)

## 🧩 PROJECT IDENTITY
- **Project Name:** Ultimate Playback
- **Project Type:** Platform / SaaS / AI Tool
- **Core Purpose:** The musician-facing playback engine for high-performance live execution of tracks, MIDI, and cues.
- **Target Users:** Live musicians and performers.
- **AI Focus:** Waveform visualization, localized haptic cues, and intelligent MIDI mapping.

## 🏗️ SYSTEM MODULES
1. **Playback Engine:** High-performance audio engine for multi-track playback.
2. **Haptic Cue Module:** Integration with mobile vibration for silent "MD-AI" cues.
3. **MIDI Sync Bridge:** Connection to UltimateSyncServer for cross-device synchronization.
4. **Dynamic Song Map:** Real-time section indicators following the global timeline.
5. **Offline Performance Core:** Reliable playback without active network connection.

## 🧠 INTELLIGENCE REQUIREMENTS
- **Silent Cues:** AI-driven haptic patterns for tempo and section transitions.
- **Bilingual MD-AI:** Support for English/Spanish voice cues during playback.
- **Latency Prediction:** Real-time monitoring of sync latency between devices.

## 🛠️ UNIVERSAL STANDARDS
- **Frontend:** React Native (Expo)
- **Audio:** Custom Native Audio Hooks / Expo AV
- **Sync:** WebSocket / MIDI over Network
- **Deployment:** Apple IPA / TestFlight
