# 🎸 ULTIMATE MUSICIAN (ADMIN APP)

## 🧩 PROJECT IDENTITY
- **Project Name:** Ultimate Musician Admin
- **Project Type:** Platform / SaaS / AI Tool
- **Core Purpose:** Management app for creating songs, stems jobs, and live performance oversight with vertical mixer controls and waveform timelines.
- **Target Users:** Music Directors, Worship Leaders, and Administrators.
- **AI Focus:** Automated stems job creation, live waveform oversight, and integration with CineStage Brain.

## 🏗️ SYSTEM MODULES
1. **Song & Stems Engine:** Module for creating new songs and triggering stem separation jobs.
2. **Vertical Mixer HUD:** Real-time vertical track faders with solo/mute capabilities.
3. **Live Waveform Navigator:** Global waveform timeline with section-based navigation (Chorus, Verse, etc.).
4. **Director HUD:** Control center for Click, Guide, and Pad toggles during performance.
5. **CineStage Sync:** Integration with the Cloudflare/Docker backend for data persistence and AI processing.

## 🧠 INTELLIGENCE REQUIREMENTS
- **Waveform Sectioning:** Automatic detection of song sections for visual navigation.
- **Stem Job Optimization:** Predictive resource allocation for audio processing tasks.
- **MD-AI Integration:** Voice cues and haptic feedback triggers for directors.

## 🛠️ UNIVERSAL STANDARDS
- **Frontend:** React Native (Expo) - SDK 51/54
- **State Management:** React Context / Hooks
- **Styling:** Tailwind CSS / Native Styles
- **Deployment:** Expo Dev Client / IPA Builds
